## Simple Calculator with Typescript + Inquirer.js

Calculator in terminal to do simple operations that I perform to test debug in *VS Code*.

## Install

Install all the dependencies:
````shell
npm install
npm install --only=dev
````
## Commands

Run without compile:
````shell
npm run ts-node
````

Compile and run:
````shell
npm run start
````

Compile:
````shell
npm run build
````
